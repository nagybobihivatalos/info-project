Bevezetés

Ez a Python script a felhasználótól kérdezi, hogy melyik hálózati eszközt szeretne csatlakoztatni. A program az eszközhöz megfelelő kábelezést javasol a felhasználónak.

Az eszközök, amikkel dolgozik:

  • Switch
  • Router
  • Hub
  • Workstation

A kód különböző feltételek alapján különböző kábeltípusokat ajánl, attól függően, hogy melyik eszközt szeretnénk csatlakoztatni.

A kód működése

  1. Felhasználói bemenet: „Mondj egy hálózati eszközt! (switch/router/hub/workstation)”
  2. Eszközkategória ellenőrzés: kisbetűs formában történik (.lower())
  3. Hibakezelés: Ha érvénytelen eszközt ad meg a felhasználó, a program kiírja:
    „Arra az eszközre nem tudok mondani sajnos.”

Lehetőségek a jövőbeli fejlesztésre

  • Bővíthető további eszközökkel (pl. modem, bridge)
  • Felhasználói input alaposabb validálása
  • Grafikus felület (GUI) integrálása
